API

AFoi desenvolvida uma API RESTful para gerenciar livros, permitindo operações de cadastro, atualização e remoção. Através de requisições HTTP (POST, PUT, DELETE), é possível inserir novos livros, editar informações existentes e excluir registros. A API utiliza JSON como formato de troca de dados. Ela foi projetada com foco na simplicidade e eficiência. A aplicação garante controle e organização do acervo de livros de forma automatizada.

Gerenciador de tarefas

Desenvolver um gerenciador de tarefas em JavaScript puro é uma ótima forma de praticar a manipulação do DOM e eventos. O sistema pode permitir criar, editar, marcar como concluídas e remover tarefas. É possível manter os dados salvos no navegador, mesmo após fechar a página. Tudo isso pode ser feito sem depender de bibliotecas ou frameworks externos. Esse projeto é ideal para reforçar conhecimentos fundamentais da linguagem.

Site em HTML

Desenvolver um site usando HTML e CSS é o primeiro passo para entrar no mundo da criação web. Com o HTML, estruturamos os conteúdos da página, como textos, imagens e links. Já o CSS permite estilizar esses elementos, definindo cores, fontes, layouts e muito mais. Juntos, eles formam a base visual e estrutural de qualquer site. Esse processo ajuda a entender como funciona a construção de páginas responsivas e bem apresentadas.
